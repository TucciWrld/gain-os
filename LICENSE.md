## How to Attribute

When using or distributing Gain OS:

1. **Include license file** - Always include this LICENSE.md
2. **Credit contributors** - See AUTHORS.md
3. **Preserve notices** - Keep all original copyright notices
4. **Disclose changes** - If modified, clearly mark changes
5. **Follow GPL terms** - If distributing, provide source code
